🛒 DigiTools – Digital Products Marketplace

A modern React-based web application where users can explore and purchase digital products like AI tools, templates, and automation kits.

📌 Description

DigiTools is a frontend e-commerce-style application focused on digital products. Users can browse products, add them to the cart, remove items, and complete checkout—all with real-time UI updates and notifications.

This project demonstrates core React concepts like state management, component structure, and user interaction handling.

🚀 Technologies Used
⚛️ React
⚡ Vite
🎨 CSS / Tailwind CSS
🔔 React Toastify
🧠 JavaScript (ES6+)
✨ Features
🔹 1. Add to Cart System
Users can add products to the cart
Prevents duplicate items
Displays instant notification
🔹 2. Remove from Cart
Remove individual items easily
Updates total price dynamically
🔹 3. Checkout System
Clears the cart on checkout
Shows success notification
🔹 4. Real-time Price Calculation
Automatically updates total cart price
Reflects add/remove actions instantly
🔹 5. Toast Notifications
Success, warning, and info messages
Clean user feedback using React Toastify
🔹 6. Component-Based Architecture
Navbar
Banner
Products Section
Pricing Section
Footer
And more…
